# 📊 HS-Statistical Assistant

**Professional Statistical Analysis Platform for Academic Researchers**

A free, browser-based statistical analysis tool designed to help researchers
analyse questionnaire/survey data quickly — without complex statistical software.

---

## ✨ Features

| Category | Analyses |
|---|---|
| 📊 Descriptive | Descriptive Statistics, Frequency Tables |
| 📐 Normality | Shapiro-Wilk, Kolmogorov-Smirnov, Anderson-Darling, QQ-Plot |
| 🔗 Correlation | Pearson, Spearman, Heatmap |
| α Scale | Cronbach's Alpha, Item-Total, Alpha-if-Deleted |
| Σ Factor | KMO, Bartlett's Test, Varimax Rotation, Scree Plot |
| t Comparative | Independent T-Test (Cohen's d, Levene), One-Way ANOVA (Tukey), Chi-Square (Cramer's V) |
| β Regression | Simple, Multiple (Standardised Beta), Logistic (Odds Ratio, ROC/AUC) |
| ⟳ Mediation | Bootstrap 1000, Sobel Test, Path Diagram |

- **Upload** Excel files (.xlsx / .xls)
- **Download** all results as **PDF reports**
- Completely **free** — hosted on Streamlit Community Cloud
